# โครงการวิทิสา 49 ล้านนาที

ระบบบันทึกและประมวลผลการปฏิบัติสมาธิวิทิสา เนื่องในโอกาสครบอายุ 49 พรรษา สมเด็จพระบรมราชินี

## จุดประสงค์

รวมพลังจิตปฏิบัติสมาธิวิทิสาจากพสกนิกรทั่วประเทศ สะสมให้ครบ **49 ล้านนาที** ภายในวันที่ **31 กรกฎาคม 2569**

## กลุ่มเป้าหมาย

- โรงเรียน / หน่วยงาน / องค์กร (สาขาเป็นผู้ประสานงานบันทึกรายวัน)
- ประชาชนทั่วไป / รายบุคคล (บันทึกเองผ่านระบบ)
- สาขาสถาบันพลังจิตตานุภาพ (~30 สาขา)

## ฟีเจอร์หลัก

### หน้าแรก
- ชื่อโครงการ + พระบรมฉายาลักษณ์
- 3 ปุ่มหลัก: **สมัคร** / **เข้าไลน์** / **บันทึกผล**
- คลิปแนะนำ: ขั้นตอนสมัคร, การทำวิทิสา (Thai/Eng), การบันทึกผล
- สถิติการสมัคร / ตัวเลขพลังจิตสะสม

### Real-time Dashboard
- **ตัวเลขสะสม** — Scrolling digit แสดงยอดนาทีรวมแบบ real-time
- **แผนที่พลังจิตทั่วไทย** — Heat Map แบ่งตามจังหวัด/สาขา
- **Progress Bar** — แถบความคืบหน้าสู่เป้าหมาย 49 ล้านนาที พร้อม milestones
- **Live Feed** — Ticker แสดงการบันทึกผลล่าสุด

### Gamification
- Daily Target: ยอดที่ต้องการในแต่ละวันเพื่อให้ทันกำหนด
- Leaderboard: อันดับสถานศึกษา / หน่วยงาน

### เพิ่มเติม
- Multi-language (ไทย / English / 中文)
- โพสต์การ์ดถวายพระพร + ร่วมส่งรูปปฏิบัติสมาธิ
- รองรับการแนบรูปจากโรงเรียน

## การบันทึกผล 2 รูปแบบ

| รูปแบบ | ผู้บันทึก | ลักษณะ |
|--------|----------|--------|
| องค์กร/โรงเรียน | สาขาผู้ประสานงาน | บันทึกยอดรวมรายวัน (คล้ายโครงการ 600 โรงเรียน) |
| รายบุคคล | ประชาชนทั่วไป | บันทึกเองผ่านระบบ |

## ความต้องการด้าน Infrastructure

ระบบต้องรองรับผู้ใช้งานจำนวนมากจากทั่วประเทศพร้อมกัน โดยใช้แนวทางจาก [Infrastructure Checklist](https://github.com/monthop-gmail/ai-skill/blob/master/infrastructure/dev-env/CHECKLIST.md) ครอบคลุม:

1. **Local Dev Architecture** — 1 service = 1 repo, Docker Compose, CF Tunnel
2. **Local to Production Sync** — compose เหมือนกันทุก env, ต่างกันแค่ `.env`
3. **Environment Separation** — DNS แยก env, git branching, branch protection, CI/CD
4. **Secret Management** — SOPS + age encryption
5. **Overall Structure** — directory layout, modular multi-service
